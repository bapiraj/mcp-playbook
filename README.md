# MCP Playbook

A collection of Model Context Protocol (MCP) servers — built to explore, learn, and share how MCP works in practice.

## Servers

| Server | Description |
|--------|-------------|
| [01-todo-mcp-server](./01-todo-mcp-server) | A simple MCP server for managing a todo list backed by a CSV file |
| [02-job-platform-database-mcp](./02-job-platform-database-mcp) | A job recruiting platform MCP server backed by SQLite — supports candidates searching/applying for jobs and recruiters managing postings and applicants |

## What is MCP?

The [Model Context Protocol](https://modelcontextprotocol.io/) is an open standard that lets AI assistants connect to external tools and data sources through a unified interface. Each server in this repo exposes tools that any MCP-compatible client (Claude Desktop, Claude Code, etc.) can use.

## Getting Started

Navigate into any server directory and follow its README for setup instructions.
